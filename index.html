O que aprendi?

A Arquitetura de Computadores do Século XXI se refere ao design e à estrutura de sistemas de computação modernos, que evoluíram para suportar as crescentes demandas de processamento e armazenamento de dados. Esse campo engloba as estruturas físicas e lógicas dos sistemas, incluindo processadores, memória, e métodos de interconexão.


Lógica de Programação é a base para a criação de algoritmos e desenvolvimento de software, pois envolve o raciocínio estruturado necessário para resolver problemas por meio de uma sequência de instruções bem definidas. Trata-se de organizar passos lógicos para atingir um objetivo, com o uso de variáveis, operadores e estruturas de controle.

O que aprendi?

Desenvolvimento Dinâmico refere-se a uma abordagem de desenvolvimento de software que prioriza a flexibilidade e a adaptação contínua às mudanças ao longo do processo de criação. Esse modelo contrasta com métodos tradicionais, como o modelo em cascata, em que o processo é rigidamente sequencial e mais difícil de alterar uma vez iniciado.


Front-End é a área do desenvolvimento web responsável pela criação da interface visual de um site ou aplicativo, ou seja, tudo aquilo com o qual os usuários interagem diretamente. É a "face" do site, sendo composta por elementos como botões, menus, textos, imagens e animações, e é essencial para proporcionar uma experiência de usuário (UX) agradável e intuitiva.


Experiência do Usuário refere-se à forma como uma pessoa se sente ao interagir com um produto, serviço ou sistema. Em desenvolvimento de software e design de interfaces, UX é focada em criar produtos que sejam úteis, fáceis de usar e que atendam às necessidades dos usuários de maneira agradável e eficiente. A UX engloba todos os aspectos da interação, desde a usabilidade e acessibilidade até as emoções e percepções dos usuários.

O que aprendi?

A Prática Integradora em Desenvolvimento de Software é uma abordagem educacional prática que visa integrar conhecimentos de diferentes áreas de um curso de desenvolvimento de software em um projeto ou atividade específica. A ideia é que, por meio da prática, os estudantes possam aplicar, consolidar e interligar o conteúdo aprendido, trabalhando em situações reais ou simuladas que envolvem planejamento, programação, testes e entregas.


A Prática Integradora em Desenvolvimento de Software é uma atividade acadêmica aplicada, que tem como objetivo conectar e consolidar o conhecimento adquirido ao longo de diversas disciplinas de um curso, colocando os alunos em contato com o processo completo de desenvolvimento de software. Ela promove uma visão integrada do que é necessário para planejar, desenvolver, testar e entregar um produto de software, sendo uma experiência crucial para a formação profissional.
